## Cluster Management

* minikube start: Starts a local Kubernetes cluster.
* minikube stop: Stops a running local Kubernetes cluster.
* minikube pause: Pauses Kubernetes without impacting current data.
* minikube unpause: Unpauses a paused Kubernetes cluster.
* minikube delete: Deletes a local Kubernetes cluster entirely.
* minikube status: Gets the status of a local Kubernetes cluster. [1, 2, 3, 4, 5] 

## Multi-Node & Profile Management

* minikube node add: Adds a new node to an existing cluster.
* minikube node delete: Removes a node from an active cluster.
* minikube node list: Lists all nodes in the current cluster.
* minikube profile list: Lists all available minikube profiles.
* minikube profile <name>: Sets the default active minikube profile. [6, 7, 8, 9, 10] 

## Configuration & Environment

* minikube config set <key> <value>: Sets a persistent configuration value (e.g., driver).
* minikube config view: Displays the active persistent configuration properties.
* minikube docker-env: Configures your terminal to point to minikube's Docker daemon.
* minikube ip: Retrieves the IP address of the specified node.
* minikube ssh: Logs into the underlying node via an interactive SSH terminal. [11, 12, 13, 14, 15] 

## Application Networking & Addons

* minikube service <name>: Connects to and opens a Kubernetes service in your browser.
* minikube tunnel: Connects to LoadBalancer services via your local machine network.
* minikube addons list: Lists all available minikube features and extensions.
* minikube addons enable <name>: Activates a built-in addon (e.g., ingress, dashboard).
* minikube dashboard: Launches the web-based Kubernetes user interface in your browser. [16, 17, 18, 19, 20] 

## Troubleshooting & Logs

* minikube logs: Returns the logs of the core cluster bootstrap system.
* minikube update-context: Synchronizes the active kubectl context to match minikube.
* minikube version: Prints the exact local minikube binary version.
